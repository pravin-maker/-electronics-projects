# 🪪 RFID Attendance System

## 📌 Description
This project is an IoT-based RFID attendance system using ESP8266. When an RFID card is scanned, the system reads the card ID and sends the data through WiFi to an online database. The person's name and the scan time are automatically recorded in an Excel sheet that can be viewed from another device.

## 🧰 Components Used
- ESP8266
- RFID Reader (MFRC522)
- RFID Cards / Tags
- WiFi Network
- Computer or cloud service for Excel logging

## ⚙️ Working Principle
When a user scans an RFID card, the RFID reader detects the unique ID of the card.  
The ESP8266 processes the data and sends it through WiFi to an online system where the name and timestamp are stored in an Excel sheet.  
This allows attendance to be monitored from another device in real time.
