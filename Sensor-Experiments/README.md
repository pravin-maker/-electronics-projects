# 🌡 Sensor Experiments

## 📌 Description
This section contains my experiments with different electronic sensors. I test and learn how sensors work, how to interface them with microcontrollers, and how to read and process sensor data.

## 🔬 What I Experiment
- Interfacing sensors with microcontrollers
- Reading sensor data
- Testing sensor behaviour
- Understanding practical applications of sensors

## 🧰 Platforms Used

- ESP8266
- Basic electronic components
