# 💡 Shift Register LED Control (74HC595 + 555 Timer)

## 📌 Description
This project demonstrates controlling multiple LEDs using a 74HC595 shift register and a 555 timer IC. The LEDs produce sequential lighting patterns based on clock pulses generated by the timer.

## 🧰 Components Used
- 74HC595 Shift Register IC
- 555 Timer IC (NE555)
- LEDs
- Resistors
- Potentiometer
- Dot PCB Board
- Power Supply

## ⚙️ Working Principle
The 555 timer IC is configured in astable mode to generate continuous clock pulses.  
These pulses are connected to the clock input of the 74HC595 shift register.

The shift register shifts the output signals sequentially, which turns on LEDs in different patterns.

A potentiometer is used to adjust the frequency of the 555 timer, allowing the speed of the LED pattern to be controlled.

## 🔧 Implementation
The circuit was assembled and soldered on a dot PCB board for a stable hardware setup.
