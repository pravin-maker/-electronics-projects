## 📌 Description
This project demonstrates an LED chaser circuit with multiple lighting effects and patterns. By controlling the LEDs sequentially, different visual patterns such as running lights, alternate blinking, and pattern shifts are created.

## 🧰 Components Used
- Arduino Uno
- LEDs
- Resistors
- Breadboard
- Jumper wires

## ⚙️ Working Principle
The Arduino program sequentially turns ON and OFF LEDs to create a chasing light effect.
