# 🔧 Device Restoration

## 📌 Description
This section contains my electronics restoration and repair projects. These projects are based on scrap electronic devices collected from local electronics markets.

## 🎯 Objective
The goal of these projects is to:
- Understand real-world electronic circuits  
- Practice hardware troubleshooting  
- Repair and restore faulty devices  
- Improve practical electronics skills  

## 🔬 What I Do
- Analyze circuit boards  
- Identify faulty components  
- Replace damaged parts  
- Re-solder connections  
- Test and verify working condition
