# 🔢 7-Segment LED Display

## 📌 Description
This project demonstrates how to control a 7-segment LED display to show numbers using basic electronic components and a microcontroller.

## 🧰 Components Used
- 7-Segment LED Display
- Microcontroller (Arduino / ESP8266)
- Resistors
- Breadboard
- Connecting wires
- Power supply

## ⚙️ Working Principle
A 7-segment display contains seven individual LED segments. By controlling these segments in different combinations, numbers from 0 to 9 can be displayed.

The microcontroller sends signals to the segments to form the required digit.
